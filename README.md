# Smart File Organizer

A beginner-friendly Python project that automatically organizes files into categories like Images, Documents, Videos, etc.

## 🚀 Features
- Automatically sorts files into folders
- Creates folders if they don't exist
- Handles common file types
- Beginner-friendly structure

## 🛠 Built With
- Python 3

## ▶️ How to Run
1. Download the project
2. Place files inside a test folder
3. Run main.py
4. Files will be organized automatically
